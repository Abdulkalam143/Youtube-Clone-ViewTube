## рҹ“Ң Overview

**ViewTube** is a production-grade YouTube clone UI built entirely with **vanilla HTML, CSS, and JavaScript** вҖ” no React, no Vue, no Tailwind, no libraries of any kind.

Every pixel, interaction, animation, and layout was handcrafted to replicate the YouTube experience, while pushing the limits of what's possible with pure web fundamentals.

> Built as part of my **#100DaysOfCode** challenge to master frontend fundamentals before moving into React & full-stack development.

## вңЁ Features

### рҹҺЁ UI & Design
- **Dark mode** (default) + **Light mode** toggle with smooth transitions
- **Glassmorphism navbar** with `backdrop-filter` blur effect
- **16 unique hand-crafted SVG thumbnails** вҖ” each themed to a content category (space, gaming, food, code, AI, travel, finance and more)

## рҹҡҖ Getting Started

No build steps. No installs. No dependencies.

```bash

git clone https://github.com/Abdulkalam143/viewtube.git

# 2. Open in browser
```

## рҹ§  What I Learned

Building this project from scratch taught me things that no tutorial could вҖ” here's what stood out:

- **CSS architecture at scale** вҖ” using CSS custom properties (`--variables`) for theming made dark/light mode trivial to implement
- **SVG as a design tool** вҖ” drawing 16 themed thumbnails entirely in SVG deepened my understanding of paths, gradients, transforms, and even SVG animations
- **DOM performance** вҖ” batch-rendering cards with `innerHTML` vs. individual `appendChild` calls, and why it matters at scale
- **State management without a framework** вҖ” a single `state` object + pure functions is surprisingly powerful and maintainable
- **Event delegation** вҖ” attaching one listener to a parent vs. hundreds of listeners on children
- **Responsive design from first principles** вҖ” building a grid that genuinely works from 320px to 1440px without a CSS framework
